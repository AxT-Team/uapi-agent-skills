# 生成摸摸头GIF

**接口地址**：`POST /image/motou`

**分类**：[Image](../resources/Image.md)

**Operation ID**：`post-image-motou`

## 这个接口适合什么时候用

除了使用QQ头像，你还可以通过上传自己的图片或提供图片URL来制作独一无二的摸摸头GIF。

## 功能概述
此接口通过POST方法，支持两种方式生成GIF：
1.  **图片URL**：在表单中提供 `image_url` 字段。
2.  **上传图片**：在表单中上传 `file` 文件。

## 使用须知
- **响应格式**：接口成功时直接返回 `image/gif` 格式的二进制数据。
- **参数优先级**：如果同时提供了 `image_url` 和上传的 `file` 文件，系统将 **优先使用 `image_url`**。
- **背景颜色**：同样支持 `bg_color` 表单字段来控制GIF背景。

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

这个接口没有显式参数，通常可以直接调用。

## 请求体

包含图片来源和背景色的表单数据。必须提供 'image_url' 或 'file' 两者之一。

**是否必填**：是

**内容类型**：`multipart/form-data`

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 生成成功！响应体是GIF格式的图片二进制数据。 |
| `400` | 请求参数错误。必须提供 'image_url' 或上传 'file' 文件两者中的一个。 |
| `500` | 服务器内部错误。可能的原因包括：从URL获取图片失败、处理上传文件失败，或在生成GIF过程中发生错误。 |
