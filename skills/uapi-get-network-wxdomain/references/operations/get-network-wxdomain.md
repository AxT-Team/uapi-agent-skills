# 检查域名在微信中的访问状态

**接口地址**：`GET /network/wxdomain`

**分类**：[Network](../resources/Network.md)

**Operation ID**：`get-network-wxdomain`

## 这个接口适合什么时候用

准备在微信里推广你的网站？最好先检查一下域名是否被“拉黑”了。

## 功能概述
这个接口可以帮你查询一个域名在微信内置浏览器中的访问状态，即是否被微信封禁。这对于做微信生态推广和营销的开发者来说至关重要。

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

| 参数名 | 位置 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| `domain` | query | string | 是 | 需要查询的域名。 |

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 查询成功！返回该域名在微信中的状态。 |
| `400` | 请求参数无效。 |
| `502` | 查询上游接口失败。 |
