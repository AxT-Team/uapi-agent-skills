# 随机字符串

**接口地址**：`GET /random/string`

**分类**：[Random](../resources/Random.md)

**Operation ID**：`get-random-string`

## 这个接口适合什么时候用

无论是需要生成一个安全的随机密码、一个唯一的Token，还是一个简单的随机ID，这个接口都能满足你。

## 功能概述
你可以精确地控制生成字符串的长度和字符集类型，非常灵活。

## 使用须知

> [!TIP]
> **字符集类型 `type` 详解**
> 你可以通过 `type` 参数精确控制生成的字符集：
> - **`numeric`**: 纯数字 (0-9)
> - **`lower`**: 纯小写字母 (a-z)
> - **`upper`**: 纯大写字母 (A-Z)
> - **`alpha`**: 大小写字母 (a-zA-Z)
> - **`alphanumeric`** (默认): 数字和大小写字母 (0-9a-zA-Z)
> - **`hex`**: 十六进制字符 (0-9a-f)

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

| 参数名 | 位置 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| `length` | query | integer | 否 | 你希望生成的字符串的长度。有效范围是 1 到 1024。 |
| `type` | query | enum: numeric, lower, upper... | 否 | 指定构成字符串的字符类型。 |

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 生成成功！ |
| `400` | 无效的请求参数。 |
| `500` | 服务器内部错误。 |
