# 查询手机归属地

**接口地址**：`GET /misc/phoneinfo`

**分类**：[Misc](../resources/Misc.md)

**Operation ID**：`get-misc-phoneinfo`

## 这个接口适合什么时候用

想知道一个手机号码来自哪里？是移动、联通还是电信？这个接口可以告诉你答案。

## 功能概述
提供一个国内的手机号码，我们会查询并返回它的归属地（省份和城市）以及所属的运营商信息。

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

| 参数名 | 位置 | 类型 | 必填 | 说明 |
|--------|------|------|------|------|
| `phone` | query | string | 是 | 需要查询的11位中国大陆手机号码。 |

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 查询成功！返回号码的归属地和运营商信息。 |
| `400` | 请求参数错误。请检查你是否提供了 `phone` 参数，以及它是否是有效的11位手机号码。 |
| `500` | 查询失败。我们的号码归属地数据库可能暂时无法访问，请稍后重试。 |
