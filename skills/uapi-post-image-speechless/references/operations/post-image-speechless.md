# 生成你们怎么不说话了表情包

**接口地址**：`POST /image/speechless`

**分类**：[Image](../resources/Image.md)

**Operation ID**：`post-image-speechless`

## 这个接口适合什么时候用

你们怎么不说话了？是不是都在偷偷玩Uapi，求求你们不要玩Uapi了

## 使用须知
- **响应格式**：接口成功时直接返回 `image/png` 格式的二进制数据。
- **文字内容**：至少需要提供 `top_text`（上方文字）或 `bottom_text`（下方文字）之一。
- **梗图逻辑**：上方描述某个行为，下方通常以「们」开头表示劝阻，形成戏谑的对比效果。

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

这个接口没有显式参数，通常可以直接调用。

## 请求体

包含表情包文字内容的JSON对象。至少需要提供上方或下方文字之一。

**是否必填**：是

**内容类型**：`application/json`

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 生成成功！响应体是PNG格式的表情包图片二进制数据。 |
| `400` | 请求参数错误。必须提供 'top_text' 或 'bottom_text' 至少其中之一。 |
| `500` | 服务器内部错误。在生成表情包图片过程中发生错误。 |
