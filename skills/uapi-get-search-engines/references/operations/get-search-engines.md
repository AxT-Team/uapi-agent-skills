# 搜索引擎配置

**接口地址**：`GET /search/engines`

**分类**：[智能搜索](../resources/智能搜索.md)

**Operation ID**：`get-search-engines`

## 这个接口适合什么时候用

获取 UAPI Pro Search 引擎的详细信息，包括支持的功能特性、参数限制和使用说明。

## 功能概述

此接口返回搜索引擎的完整配置信息，你可以用它来：
- 了解搜索引擎支持哪些功能（如站内搜索、文件类型过滤等）
- 获取参数的默认值和限制范围
- 查看当前引擎版本和可用状态

适合在应用初始化时调用，或用于动态配置搜索界面。
      

## 调用前检查

- 先确认用户真正需要的是最终结果，而不是某个中间步骤。
- 如果参数说明里写了互斥、默认值或生效条件，请严格按说明组织请求。
- 如果用户没有提供必要参数，先补齐参数再调用，不要靠猜。

## 参数

这个接口没有显式参数，通常可以直接调用。

## 响应码

| 状态码 | 说明 |
|--------|------|
| `200` | 成功返回搜索引擎的详细信息 |
